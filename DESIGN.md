# Design System

A premium sports mobile application.

## Colors
- Primary: Turquoise/Cyan (#00E5FF)
- Background: Graphite Black (#0F1115)
- Surface: Dark Grey (#1C1E24)
- Text: White

## Typography
- Font Family: Inter

## Style
- Dark mode only
- Modern sports app look
- Clean and highly visual
- Smooth rounded corners
- Glassmorphism effects
